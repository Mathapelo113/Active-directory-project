# Active-directory-project# 

## Project Overview

This project documents my understanding of Active Directory concepts through Microsoft Learn and technical training resources.

## What I Learned

### What is Active Directory?

Active Directory is Microsoft's directory service that allows organizations to centrally manage users, computers, passwords, and security policies.

### What is a Domain?

A domain is a collection of users, computers, and resources managed under one network.

### What is a Domain Controller?

A Domain Controller is a server that stores Active Directory information and verifies user logins.

### Users and Groups

Users represent employees. Groups are used to manage permissions more efficiently.

### Organizational Units (OU)

Organizational Units are containers used to organize users and computers within a domain.
